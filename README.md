# QRコード一括作成ツール

このツールは、テキストファイルに書かれた複数の文字列を一括で読み込み、中央に文字が印字された**QRコード画像**を自動生成する単体アプリです。
事前に指定された環境（Node.jsなど）に依存せず、実行ファイル（.exe）のみで動作します。

## ⚙️ 初回の準備（ビルド方法）

プログラムの実行ファイル（.exe）を作成します。
お使いのターミナル（PowerShellやコマンドプロンプトなど）でこのフォルダを開き、以下のコマンドを**ご自身で**実行してください。

```powershell
cargo build --release
```

※ライブラリのダウンロードやコンパイルが行われるため、初回のビルドには少し時間がかかります。
ビルドが完了すると、以下の場所に実行ファイルが作成されます。

👉 **`target\release\qr-batch-generator.exe`**

---

## 🚀 アプリの使い方

ビルド完了後、作成された `qr-batch-generator.exe` を使ってQRコードを一括生成します。
この `.exe` ファイルは、コピーしてデスクトップなどの別のフォルダに移動しても単体で動きます。

### 1. `input.txt` の設定
`qr-batch-generator.exe` と同じフォルダに **`input.txt`** という名前のファイルを用意します。
（※初回実行時にファイルが存在しない場合は、サンプルの `input.txt` が自動で作られます）

このファイル内に、QRコード化したい文字列を「1行ずつ」入力して保存してください。

**`input.txt` の入力例:**
```text
https://example.com/001
文字列サンプル123
社員番号A-001
```

### 2. QRコードの一括生成
`qr-batch-generator.exe` をダブルクリック（またはコマンドから実行）します。

### 3. 出力結果の確認
同じフォルダ内に **`output`** というフォルダが自動で作成され、その中に `qr_001.png`, `qr_002.png` のように画像ファイルが一括保存されます。

---

## 補足事項
* このツールは Rust 言語で作成されています（コアファイル: `src/main.rs`, `Cargo.toml`）。
* 出力画像の中心に描画されるテキストが長すぎる場合（18文字以上）、レイアウト崩れを防ぐため画像上の印字は末尾が「…」と省略されます。（※カメラ等で読み取るQRコードの中身のデータ自体は、省略されず全ての文字が保持されます）
* 画像上の日本語テキストは、Windows標準のフォント（メイリオ または MSゴシック）を使用して綺麗に描画されます。
